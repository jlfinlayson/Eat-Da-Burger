# EAT DA BURGER

EAT DA BURGER is a Bob's Burger themed app that allows users to create and eat the iconic burgers featured in the show. <br>

 
 ### Technologies Used

* Express
* SQL
* Heroku
* Handlebars
* Node.js
* Javascript
* NPM
* SQL
* Path
* dotenv
